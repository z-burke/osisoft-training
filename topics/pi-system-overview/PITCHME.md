### What is the PI System?
@snap[content]
@snap[content-10]
The PI System is a @css[green-emphasis](data infrastructure). It
@ul[text-08](false)
- **collects** data,
- **historizes** data,
- **contextualizes** data,
- **analyzes** data,
- and **visualizes** data
@ulend
in meaningful ways in order to support decision making.
